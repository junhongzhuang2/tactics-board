# 飞盘战术板 · 使用指南

一份给教练 / 队员看的简单上手说明。很多功能藏在**双击、右键、拖拽**里——文末有一张「隐藏操作速查表」，记不住就翻最后。

---

## 一、第一个页面：战术板列表

打开网站看到的就是这里。

| 想做什么 | 怎么做 |
|---|---|
| 新建战术板 | 点黄色虚线卡片 **＋ 新建战术板** |
| 打开战术板 | 点任意一张战术板卡片 |
| 改名字 | 卡片右侧 **重命名** |
| 删除 | 卡片右侧 **删除** |

---

## 二、移动球员和飞盘

进入战术板后，场上有 **红队 7 人 + 蓝队 7 人 + 1 个飞盘**。

- **移动**：直接用鼠标拖球员或飞盘。
- **加一个飞盘**：顶栏 **+ 盘**。
- **删一个飞盘**：**右键** 那个飞盘。
- **给球员改名 / 开视野锥**：**双击** 该球员，弹出小面板。

---

## 三、视野锥（球员朝向）

想标出某个球员面朝哪边：

1. **双击** 球员 → 勾选 **显示视野锥**。
2. 球员旁会出现一个**旋转手柄**，**拖**它就能转动朝向。

---

## 四、曲线轨迹（让跑动走弧线）

默认两帧之间是直线移动。想让它走弧线：

1. 先在某一帧把球员/飞盘**拖动**一下（制造移动）。
2. **单击**这个移动过的元素 → 旁边浮现 **曲线** 按钮。
3. 点 **曲线** → 出现一个手柄，**拖**它把直线掰成弧线。
4. **双击**手柄 → 取消曲线，恢复直线。

---

## 五、阵型预设（一键摆位）

底栏 **阵型 ▲** → 选一个：**默认 / 竖排 / 横排 / Zone / Junk**。

- 选中后整场 14 人 + 飞盘一键到位。
- 约定：**红队（左）进攻、蓝队（右）防守、飞盘给红队**。
- 只改**当前这一帧**，可以撤销。
- 播放中、或时间轴停在两帧中间时，按钮会变灰不可用（停到某一帧上才行）。

---

## 六、标注（箭头 / 形状 / 文字）

左上角 **✎ 标注** 展开工具栏。

**工具**：选择 · 传盘（虚线箭头）· 跑位（实线箭头）· 矩形 · 椭圆 · 文字
**颜色**：黄 · 红 · 蓝 · 白
**作用域**：
- **本帧** = 只在这一帧显示
- **全局** = 每一帧都显示

| 想做什么 | 怎么做 |
|---|---|
| 画箭头 / 形状 | 选好工具，在场上**拖** |
| 写文字 | 选 **文字** → 点一下场地 → 打字 |
| 改文字内容 | **双击**那段文字 |
| 移动标注 | 选「选择」工具后**拖**它 |
| 改大小 | 拖标注上的小方块**句柄** |
| 删除标注 | **右键**它；或选中后按 **Delete** |
| 切换 本帧/全局、删除 | 选中标注后会浮现一个小工具条 |
| 收起整个工具栏 | 点 **«** |

---

## 七、做动画（关键帧）

核心思路：**每一帧 = 全场的一个定格画面**，播放时所有人从这一帧的位置平滑滑到下一帧。

底部**时间轴**上每个方块就是一帧。

| 想做什么 | 怎么做 |
|---|---|
| 加下一帧 | 时间轴 **＋**（在当前帧后插入，新帧会复制当前画面，再拖球员做下一个动作即可） |
| 跳到某一帧 | 点那个方块 |
| 删一帧 | **右键**那个方块 |
| 改某帧停留时长 | **拖**方块**右边缘**，或用 **时长** 输入框 |
| 预览任意时刻 | 点/拖时间轴 |
| 播放 / 暂停 | ▶ / ⏸ |
| 上一帧 / 下一帧 | ⏮ / ⏭ |
| 循环播放 | 🔁 |

> 小贴士：播放到结尾停下后，再点 **＋** 会正确地加在**最后一帧后面**。

---

## 八、撤销 / 重做 & 快捷键

| 操作 | 快捷键 |
|---|---|
| 撤销 | 顶栏 ↶ 或 **Ctrl+Z**（Mac ⌘+Z） |
| 重做 | 顶栏 ↷ 或 **Ctrl+Y** / **Ctrl+Shift+Z** |
| 删除选中的标注 | **Delete** 或 **Backspace** |
| 改战术板名字 | **双击**顶栏标题 |

---

## 九、保存与分享

- **自动保存**：改动会自动存到你这台电脑的浏览器里，顶栏显示「已保存」。
- **分享**：目前用**录屏**——把动画播放一遍录下来发给别人最直观。
- ⚠️ **注意**：数据存在**本机浏览器**里。换电脑、清浏览器缓存会丢失，重要内容请及时录屏备份。

## 🔑 隐藏操作速查表（最常忘的）

| 操作 | 作用 |
|---|---|
| **双击** 球员 | 改名字 + 开关视野锥 |
| **双击** 顶栏标题 | 改战术板名 |
| **双击** 文字标注 | 重新编辑文字 |
| **双击** 曲线手柄 | 取消曲线 |
| **右键** 飞盘 | 删除该飞盘 |
| **右键** 标注 | 删除该标注 |
| **右键** 时间轴方块 | 删除该帧 |
| **单击** 移动过的元素 | 浮现「曲线」按钮 |
| **拖** 球员 / 飞盘 | 移动 |
| **拖** 旋转手柄 | 改朝向 |
| **拖** 标注 / 句柄 | 移动 / 改大小 |
| **拖** 时间轴方块右边缘 | 改该帧时长 |
| **Ctrl+Z / Ctrl+Y** | 撤销 / 重做 |
| **Delete** | 删除选中的标注 |
